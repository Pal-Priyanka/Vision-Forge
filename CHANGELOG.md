# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-19

### Added
- **Monorepo Structure**: Organized codebase into `apps/`, `core/`, and `infrastructure/` for better scalability.
- **Production DevOps**: Multi-stage Dockerfiles and GitHub Actions pipelines.
- **Elite Documentation**: Comprehensive ARCHITECTURE, API, and SECURITY guides.
- **Dual-Model Arena**: Integrated YOLOv5 and DETR side-by-side inference.
- **Advanced Analytics**: Real-time PR curves and mAP telemetry.

### Changed
- **Styling**: Refined UI with Glassmorphism and premium design system.
- **API Engine**: Transitioned to asynchronous FastAPI execution.

### Fixed
- **Port Conflicts**: Implemented dynamic port binding and cleanup logic.
- **Build Errors**: Resolved Tailwind CSS compilation and asset mapping issues.
